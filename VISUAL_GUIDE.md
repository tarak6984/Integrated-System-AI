  duration: 0.6s

Stagger Children:
  Each child delayed by 0.1s - 0.2s

### Hover Animations
```javascript
Cards:
  - Translate Y: -8px
  - Shadow: Increase
  - Border: Color change

  - Scale: 1.05
  - Shadow: Increase

  - Scale: 1.1
  - Rotate (some)
```

```css

  - Up and down motion
  - 6s duration
  - Infinite loop
  - Ease-in-out

  - Position shift
  - 8s duration
  - Linear infinite
```

## üì± Responsive Breakpoints

### Mobile (< 768px)
```
- Single column layout
- Hamburger menu
- Larger touch targets
- Stacked buttons
- Reduced padding
- Smaller font sizes
```

### Tablet (768px - 1024px)
```
- 2-column grids
- Expanded navigation
- Medium padding
- Adjusted font sizes
```

### Desktop (1024px+)
```
- Multi-column grids
- Full navigation bar
- Maximum content width
- Larger font sizes
- Hover effects enabled
```
