## ğŸ“¦ Installation & Setup

### **Prerequisites**
- Node.js 20.0.0 or higher
- npm 10.0.0 or higher
- Git

### **1. Clone Repository**
```bash
git clone https://github.com/tarak6984/Integrated-System-AI.git
cd Integrated-System-AI
```

### **2. Install Dependencies**
```bash
npm install
```

### **3. Environment Configuration**

Create a `.env.local` file in the root directory:

```env
# Groq AI Configuration
VITE_GROQ_API_KEY=your_groq_api_key_here

# EmailJS Configuration
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key

# News API Configuration
VITE_NEWS_API_KEY=your_news_api_key

# Cloudinary Configuration (Optional)
VITE_CLOUDINARY_CLOUD_NAME=your_cloud_name
VITE_CLOUDINARY_API_KEY=your_api_key
VITE_CLOUDINARY_API_SECRET=your_api_secret
```

### **4. Get API Keys**

#### **Groq API (Required for AI Chatbot)**
1. Visit [https://console.groq.com](https://console.groq.com)
2. Sign up for a free account
3. Navigate to API Keys section
4. Create a new API key
5. Copy and paste into `.env.local`

#### **EmailJS (Required for Contact Form)**
1. Visit [https://www.emailjs.com](https://www.emailjs.com)
2. Sign up for a free account
3. Create an email service (Gmail, Outlook, etc.)
4. Create an email template
5. Get your Service ID, Template ID, and Public Key
6. Add to `.env.local`

**EmailJS Setup Guide:** See `EMAILJS_SETUP_GUIDE.md` for detailed instructions.

#### **News API (Optional)**
1. Visit [https://newsapi.org](https://newsapi.org)
2. Sign up for a free account (100 requests/day)
3. Get your API key
4. Add to `.env.local`

#### **Cloudinary (Optional)**
1. Visit [https://cloudinary.com](https://cloudinary.com)
2. Sign up for a free account
3. Get your cloud name, API key, and API secret
4. Add to `.env.local`

**Cloudinary Setup Guide:** See `CLOUDINARY_FINAL_SETUP.md` for detailed instructions.

### **5. Run Development Server**
```bash
npm run dev
```

Visit `http://localhost:5173` in your browser.

### **6. Build for Production**
```bash
npm run build
```

### **7. Preview Production Build**
```bash
npm run preview
```

## ğŸ› Troubleshooting

### **Build Errors**
```bash
# Clear cache and rebuild
rm -rf node_modules package-lock.json
npm install
npm run build
```

### **Environment Variables Not Working**
- Ensure `.env.local` exists in root directory
- All Vite env vars must start with `VITE_`
- Restart dev server after changing env vars

### **API Errors**
- Check API keys are valid
- Verify API rate limits haven't been exceeded
- Check browser console for detailed errors

### **Video Not Playing**
- Ensure video files are in `public/` directory
- Check video file formats (MP4, WebM supported)
- Verify video codecs are compatible
